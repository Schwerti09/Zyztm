# Produkt-Videos

Platziere hier kurze Hover-Videos für die Produktkarten.

**Benötigte Dateien:**
- `produkt1.mp4` → Voice Pack
- `produkt2.mp4` → Zyztm Bro
- `produkt3.mp4` → Card Booster Pack
- `produkt4.mp4` → Soundboard Pack
- `produkt5.mp4` → VIP Membership
- `produkt6.mp4` → Gaming Bundle

**Empfehlungen:**
- Format: MP4 (H.264, web-optimiert)
- Dauer: 2–3 Sekunden (loop-fähig)
- Auflösung: 720p reicht
- Inhalt: Kurze Clips aus deinen Streams passend zum Produkt
