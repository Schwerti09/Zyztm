# Intro Video

Platziere hier dein Intro-Video als `intro.mp4`.

**Empfehlungen:**
- Format: MP4 (H.264, web-optimiert)
- Auflösung: 1080p
- Dauer: ~10 Sekunden
- Inhalt: Dein Logo + Fortnite-Szenen + Neon-Overlay + "ZYZTM NEXUS" + "JOJOJO"

Du kannst ein Video mit [Canva](https://canva.com), [Pictory](https://pictory.ai) oder [Runway ML](https://runwayml.com) erstellen.
