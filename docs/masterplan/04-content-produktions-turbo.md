### Schritt 5: Discord

## Prompt-Kette für einen Artikel von Brief bis Veröffentlichung

**Prompt:**
```
Erstelle einen Content-Brief für einen Fortnite Guide über [THEMA].

Der Brief muss enthalten:
- Ziel-Keyword
- Zielgruppe
- Such-Intent
- Wettbewerb
- Empfohlene Länge

```

### Schritt 2: Outline erstellen (10 Min)
**Prompt:**
```
Basierend auf dem Brief, erstelle eine detaillierte Outline für den Artikel.

Die Outline muss enthalten:
- H1 mit Hook
- 8 H2-Sektionen mit Leitfragen

```

### Schritt 3: Content generieren (30 Min)
**Prompt:**
```
Schreibe den kompletten Artikel basierend auf der Outline.

Anforderungen:
- 1.500+ Wörter
- Deutschsprachig
- Fortnite-expert Tone
- SEO-optimiert
- Interne Links an markierten Stellen
- Featured Snippet am Anfang
```

### Schritt 4: SEO-Optimierung (10 Min)
**Prompt:**
```
Optimiere den Artikel für SEO.

Prüfe:
- Title-Tag (60 chars max)
- Meta-Description (160 chars max)
- H1 exakt einmal
- Interne Links (mindestens 3)

```

### Schritt 5: Qualitätssicherung (15 Min)
**Manuelle Prüfung:**
- Fakten-Korrektheit
- Fortnite-Expertise
- Lesbarkeit
- Grammatik
- Formatierung

### Schritt 6: Veröffentlichung (5 Min)
**Manuelle Schritte:**
- Bilder hochladen
- Tags hinzufügen
- Kategorie zuweisen
- Veröffentlichen
- Social Media teilen

# [5] QUALITÄTSSICHERUNGS-CHECKLIST

## SEO-Checks (5 Punkte)

1. **[ ] Title-Tag optimiert** (60 chars max, Haupt-Keyword vorne)
2. **[ ] Meta-Description optimiert** (160 chars max, CTA am Ende)
3. **[ ] H1 exakt einmal** (keine duplizierten H1)
4. **[ ] Interne Links gesetzt** (mindestens 3 relevante interne Links)
5. **[ ] Schema Markup** (Article Schema + FAQPage Schema)

## Inhalt-Checks (5 Punkte)

6. **[ ] Fakten-Korrektheit** (alle Fortnite-Fakten überprüft)
7. **[ ] Fortnite-Expertise** (korrekte Terminologie, Meta-Knowledge)
8. **[ ] Lesbarkeit** (kurze Sätze, klare Struktur)
9. **[ ] Grammatik** (keine Rechtschreibfehler)
10. **[ ] Formatierung** (korrekte H-Struktur, Absätze)

## Technische Checks (5 Punkte)

11. **[ ] Alt-Text für alle Bilder** (beschreibend, Keyword-relevant)
12. **[ ] Canonical URL** (korrekt, keine Duplicate Content)
13. **[ ] Mobile-Test** (responsiv, Touch-optimiert)
14. **[ ] Ladezeit-Check** (LCP < 2.5s, alle Bilder optimiert)
15. **[ ] Broken Links** (keine toten Links)

## User-Experience-Checks (5 Punkte)

16. **[ ] Featured Snippet** (40-55 Wörter, Frage-Antwort-Format)
17. **[ ] CTA vorhanden** (klar, relevant, prominent)
18. **[ ] SAC-Code Erinnerung** (strategisch platziert)
19. **[ ] Related Content** (verwandte Artikel verlinkt)
20. **[ ] Social Share** (Twitter, Discord, Reddit Buttons)
